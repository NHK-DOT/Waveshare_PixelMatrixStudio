# 像素矩阵动图工具

这是面向单块微雪 64x64 HUB75 RGB 像素矩阵的 Windows 原生 C++ 小工具。

## 编译与运行

双击 `PixelMatrixStudio.exe` 即可运行。需要重新编译时，在此目录运行 `build.bat`；它使用已安装的 MinGW `g++` 生成 EXE。

## 使用流程

1. 点击“导入图片 / GIF”，选择 GIF、PNG、JPG、BMP 或 WEBP。
2. 在“裁剪比例”输入 `16:9`、`1:1`、`4:3` 或小数比例。导入后左侧原图会出现裁剪框，框外区域以灰色显示。
3. 在裁剪框内拖动可以移动选区；从裁剪框外拖动可重新拉出选区。点击“重置居中裁剪框”可回到当前比例的默认选区。
4. 输入像素宽度和高度，例如 `64 x 32`；再用“画布 X / Y”把它布置在 64x64 的物理矩阵中，例如 `X=0`、`Y=16`。
5. 选择降采样方式：`清晰最近邻`保留硬朗像素边缘；`多点降采样（保留原色）`会为每颗 LED 采集对应原图区域的多个颜色样本，不会擅自改变色调，默认推荐使用。
6. 亮度、对比度和饱和度均可通过滑块或直接输入 `-100` 至 `100` 调整；默认均为 `0`，即完全保留原图色彩。
7. 裁剪、画布位置、像素尺寸和增强参数修改后会立即反映在右侧预览。预览中的一个方格就是 RGB 矩阵上的一颗 LED。
8. 按需导出像素 GIF、首帧 PNG，或用于 ESP-IDF 固件播放的 `.pmx` 内容文件。导出时会处理 GIF 的完整帧序列。

## PMX 内容

导出的 `.pmx` 使用 64x64 RGB332 格式，既可保存静态图片，也可保存 GIF 的完整动画和帧延时。烧录方法见 `..\matrix_idf\main\PMX烧录说明.md`。

`gif.h` 是 charlietangora/gif-h 的公共领域单头文件 GIF 编码器。
